# Rust-Raid-Assistant
This is a program to help Rust players decide if they should raid an enemy base, how to raid it, and how much boom they will need. The boom per doors file is a comma-delimeted txt file that the main script parses for data.
•	Command-line decision support application that analyzes raid viability in Rust based on gear advantage, distance risk, and base defenses
•	Data-driven calculation system that parses external configuration files to determine required explosives for different base structures
